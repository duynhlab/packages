duynhlab packages

YUM repository for the duynhlab e-commerce platform. Hosted on GitHub Pages, backed by the duynhlab/packages repository.

Install (Rocky / Alma / RHEL 9)

sudo curl -fsSL -o /etc/yum.repos.d/duynhlab.repo https://duynhlab.github.io/packages/duynhlab.repo
sudo dnf install -y duynhlab

Start the platform

The database bootstraps itself — a one-shot unit creates every per-service DB + roles and runs migrations before the backends start. Same-host PostgreSQL works out of the box (local peer auth); just enable the target:

sudo systemctl enable --now duynhlab-platform.target

For a remote or managed database, point the bootstrap at it first:

sudo cp /opt/duynhlab/etc/bootstrap.env.example /etc/duynhlab/bootstrap.env
sudo chmod 0640 /etc/duynhlab/bootstrap.env
sudoedit /etc/duynhlab/bootstrap.env   # set SUPERUSER_DSN=postgresql://postgres:...@DBHOST:5432/postgres
sudo systemctl enable --now duynhlab-platform.target
